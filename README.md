# Commit

A CMS for teams whose websites live in code.

Commit sits directly on top of GitHub. Connect your repos, open any markdown file, and edit it like a doc — no terminal, no raw markdown, no asking an engineer to push your changes.

## How it works

- **Notion-like editor** for markdown files — headers, bullets, tables, images without touching the file system
- **Real Git workflows under the hood** — Save to commit directly, or open a PR to keep your review process intact
- **Document metadata** — edit frontmatter directly in the editor, no YAML required
- **Comments** — leave feedback on docs without leaving the editor
- **GitHub stays your source of truth** — everything links back

## Getting Started

Connect your GitHub account at [usecommit.com](https://usecommit.com) and start editing.
